How to Create Rest WebServices:

Requirements:

  • Spring Tool Suite or Eclipse(with Spring boot starter pack)
  • PostMan(Rest Client to test Services )

Main Files:

  • ServiceModel.java
  • ServiceInterface.java
  • ServiceImpl.java
  • ServiceApplication.java

Steps:

  1. We need to create the model file which has all the attributes of the Service.

For example:

    • Passenger rest application which as methods like get the list of passengers and add new passengers to the list.
    • Lets, call the model file as Passenger.java
    • This class has all the attributes of passenger, for example Name, Id, phoneNumber etc.
    • Also, this class has setters and getters for the attributes which are used to get the attributes from the list of passengers.
